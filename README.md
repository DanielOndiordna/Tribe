# Tribe
Tribe CRM tools

Dankzij deze plugin zijn er diverse tools om Tribe een beetje beter te maken.
Je kunt zelf kiezen welke tools je wilt activeren. Kijk hiervoor in Tribe onder jer persoonlijke instellingen.

Let op: De keuzes die je instelt voor deze tools worden alleen opgeslagen in dit browser profiel en worden niet bewaard in Tribe.
